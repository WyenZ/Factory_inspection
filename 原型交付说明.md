# 免验厂原型交付说明

## 交付内容

本地已保存三套静态原型：

| 原型 | 目录 | 重点 |
| --- | --- | --- |
| 申请验厂入口 | `prototype/` | 从品牌中心供应商列表进入“申请验厂”，选择普通验厂或免验厂，并填写免验厂条件材料 |
| 分配验厂人员 | `prototype2/` | 在供应商中心待处理页点击“分配”，选择“是否免验：是”，查看免验厂资料/API 查询结果 |
| 待审核页 | `prototype3/` | 直接展示免验厂进入待审核后的列表效果和审核弹窗 |

## 演示重点

### prototype

- 供应商列表操作下拉包含“申请验厂”。
- 申请验厂弹窗支持普通验厂和免验厂。
- 选择免验厂后可多选 1-1、1-2、1-3、1-4 条件。
- 不同条件展示不同材料、日期、账期或企查查查询结果。

### prototype2

- 复刻供应商中心“品牌验厂管理”待处理页。
- 点击“分配”后展示“是否免验”。
- 选择“是”后折叠普通验厂时间/人员字段。
- 展示免验厂资料和 API 查询结果，便于说明待处理节点如何识别免验资料。

### prototype3

- 首屏直接进入“待审核(6)”页，避免依赖跨页面跳转。
- 仅保留两条免验厂样例：
  - 单条件：`免验厂` + `三方验厂`
  - 双条件：`免验厂` + `上市公司` + `实缴资本`
- 其他数据按普通验厂展示，不显示免验厂标签。
- 免验厂行自动展示：
  - 验厂人员：`免验厂`
  - 验厂分数：`100`
  - 验厂报告：相关证明文件
- 验厂报告列每行只显示一个蓝色文件图标；多个证明文件名并列展示在图标下方。
- 点击“审核”打开原有风格弹窗，验厂结果下拉为 `通过`、`不通过`。
